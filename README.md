# VALORANT-CLEANER

Hey everyone, what's up!

So, here's the deal with this VGC Cleaner – I made this little tool to give your PC a good scrub when it comes to Valorant and Vanguard traces.

If you've gotten banned and are planning to use a spoofer, or you just want to wipe the slate clean to try a new account, this thing helps a lot.

What does it delete?
The game folders and Riot stuff (yes, it deletes the actual Valorant installation, so you'll have to download it again afterward).

System caches, logs, and temporary files.

The registry keys that Vanguard uses to identify your machine.

Your execution history (UserAssist).

The MachineGuid (that unique ID Windows creates for your system).

Want to go even deeper?
Depending on the mode you pick, it can also:

Clear out Windows Event Logs (System, Application, Security).

Flush your network stuff (DNS, ARP, NetBIOS).

Remove firewall rules that have "Riot" in the name.

Clean up your hosts file (removing lines with riotgames, valorant, or vanguard).

Force ownership of protected folders (takeown + icacls) to make sure everything gets deleted.

How to use it?
It's pretty simple:

Download the .exe and run it as Administrator (it won't work otherwise).

When it opens, pick a mode:

1 - Quick Clean: Just the basics (folders, registry, caches).

2 - Deep Clean: Does everything I mentioned above (I recommend this one).

3 - Custom Clean: You get to choose which heavy tasks you want to run.

Wait for it to finish (there's a progress bar and a list of steps on the screen).

Once it's done, restart your PC. This is important to apply the changes, especially the MachineGuid reset.

Just a heads up:
This program will delete your Valorant. If you don't want to lose the game, just comment out the lines that delete the C:\Riot Games and Program Files folders before compiling.

Your personal files (pictures, documents, etc.) are safe – it doesn't touch those.

It runs on Windows 10/11 64-bit and needs admin rights.

That's it! Use it wisely, and good luck out there.
